README
=======
How to run:
-->Run cmd pip3 install -r requirements.txt
--> Download GoogleNews-vectors-negative300.bin in same folder as python files

For  twitter_dataset
Dataset  Required: Extract Twitter 15 and Twitter 16 Dataset in same folder as the twitter_dataset.py  file

To Run: python3 twitter_dataset.py

For FakeNews Net Dataset:
Dataset Required: gossip cop and politifact , should be extracted in same folder as code files

To Run: python3 Fakenews_preprocess.py
		python3	train.py
		python3 test.py
		python3 test_util.py
		
		
	**Accuracy will be printed on test_util.py file
